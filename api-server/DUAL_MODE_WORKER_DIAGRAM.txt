SCENARIO: User requests to create a configuration file

1. User Request:
   POST /api/v1/agent-posts
   {
     "title": "Create Config",
     "content": "file://create /tmp/config.json",
     "userId": "user123",
     "metadata": {
       "task_type": "file_operation",
       "params": {
         "operation": "create",
         "path": "/tmp/config.json",
         "content": "{\"env\": \"production\"}"
       }
     }
   }

2. Ticket Created:
   work_queue.id = 42
   work_queue.status = 'pending'
   work_queue.post_content = "file://create /tmp/config.json"
   work_queue.post_metadata = { task_type: "file_operation", ... }

3. Orchestrator Detects:
   SELECT * FROM work_queue WHERE status = 'pending' ORDER BY priority DESC

4. Worker Spawned:
   worker-1234567890-abc123 → ticket #42

5. Task Detection:
   TaskTypeDetector.detect() → { type: 'file_operation', params: { ... } }

6. Execution:
   FileOperationExecutor.execute()
     → validatePath("/tmp/config.json") ✓
     → fs.writeFileSync("/tmp/config.json", content) ✓
     → { success: true, path: "/tmp/config.json", bytes: 23 }

7. Ticket Updated:
   work_queue.status = 'completed'
   work_queue.result = {
     "task_type": "file_operation",
     "success": true,
     "operation": "create",
     "path": "/tmp/config.json",
     "bytes_written": 23
   }
   work_queue.completed_at = '2025-10-13 12:34:56'

8. Worker Terminates:
   Worker cleaned up, orchestrator continues polling
