## Function Under Test

### Function Signature
```typescript
getHookContent(content: string, title?: string): string
```

### Current Implementation Logic

1. **Title Matching & Removal** (if title provided):
   - Skip HTML comments at the start
   - Check if first non-comment line is a markdown heading matching title
   - If match found (case-insensitive), skip to first non-empty line after title

2. **Content Extraction**:
   - Split content by sentence boundaries using regex: `(?<=[.!?])\s+`
   - Extract first sentence

3. **URL Preservation**:
   - If first sentence contains URL, return it
   - Otherwise, check next 2 sentences for URLs and append if found

4. **Truncation** (if length > 300 chars):
   - Truncate while attempting to preserve URL context
   - Keep ~100 chars before and after URL
