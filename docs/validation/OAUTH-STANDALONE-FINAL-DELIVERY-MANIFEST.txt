🐛 WHAT WE'RE TESTING FOR
════════════════════════════════════════════════════════════════════════════════

Known Issue: OAuth 500 Error on Message Send
  
  Symptom:
    OAuth users receive 500 Internal Server Error when sending DM messages
  
  Root Cause:
    Worker queue caches authentication context from previous API key requests,
    causing subsequent OAuth requests to be processed with wrong credentials
  
  Evidence to Capture:
    ✅ Screenshot showing 500 error in UI
    ✅ Network log showing OAuth token in request headers
    ✅ Network log showing API key error in response body
    ✅ Control tests (API Key/PAYG) working normally
    ✅ Visual comparison between auth types
  
  This Test Suite Provides:
    • Complete visual evidence (30+ screenshots)
    • Full network traffic logs (3 JSON files)
    • Side-by-side comparison (OAuth vs controls)
    • Reproducible test case
    • Detailed error documentation
