SOLUTION:
┌────────────────────────────────────────────────────────────────────────────┐
│ Use relative URLs that Vite dev server proxy handles automatically:       │
│                                                                            │
│ BEFORE: const url = `${API_BASE_URL}${endpoint}`;                         │
│         Result: http://localhost:3000/api/activities (FAILS)              │
│                                                                            │
│ AFTER:  const url = endpoint.startsWith('http')                           │
│                    ? endpoint                                             │
│                    : `${API_BASE_URL}${endpoint}`;                        │
│         Result: /api/activities (SUCCESS via proxy)                       │
