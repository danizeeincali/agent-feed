## Request Flow Example: getHealth()

1. Component calls: monitoringApiService.getHealth()
   ↓
2. Service checks cache (5s TTL)
   ├─ Cache hit → Return cached data ✓
   └─ Cache miss → Continue
   ↓
3. Create AbortController for timeout
   ↓
4. Make HTTP GET request to /api/monitoring/health
   ↓
5. Retry logic (if needed):
   - Attempt 1: Immediate
   - Attempt 2: Wait 1s
   - Attempt 3: Wait 2s
   - Attempt 4: Wait 4s
   ↓
6. Response received
   ├─ Success (200)
   │  ├─ Parse JSON
   │  ├─ Cache result
   │  └─ Return data
   └─ Error (4xx/5xx)
      ├─ Parse error message
      ├─ Transform to Error object
      └─ Throw error
   ↓
7. Component receives HealthStatus data
   ↓
8. Render UI with status

## Cache Strategy

┌────────────────────────────────────────────────────────────────┐
│  Cache Key: /health                                            │
│  TTL: 5000ms                                                   │
│  ┌──────────────────────────────────────────────────────────┐ │
│  │ Timestamp: 1234567890                                    │ │
│  │ Data: { status: 'healthy', ... }                         │ │
│  │ Expiry: 1234572890                                       │ │
│  └──────────────────────────────────────────────────────────┘ │
└────────────────────────────────────────────────────────────────┘
         │
         │ Next request within TTL
         ▼
    Return cached data (95% faster)
         │
         │ Request after TTL
         ▼
    Fetch fresh data, update cache

## Error Handling Flow

Request → Try Fetch
           │
           ├─ Success → Return data
           │
           └─ Error
               │
               ├─ Network Error
               │  └─ Retry (3 attempts)
               │
               ├─ Timeout
               │  └─ Retry (3 attempts)
               │
               ├─ HTTP 4xx
               │  └─ No retry (client error)
               │
               ├─ HTTP 5xx
               │  └─ Retry (3 attempts)
               │
               └─ Parse Error
                  └─ No retry (invalid response)

## Integration Pattern

```typescript
// 1. Import service
import { monitoringApiService } from './services/MonitoringApiService';
import type { HealthStatus } from './services/MonitoringApiService';

// 2. Use in component
const [health, setHealth] = useState<HealthStatus | null>(null);

useEffect(() => {
  async function loadHealth() {
    const data = await monitoringApiService.getHealth();
    setHealth(data);
  }
  loadHealth();
  
  // 3. Cleanup
  return () => monitoringApiService.destroy();
}, []);
```
