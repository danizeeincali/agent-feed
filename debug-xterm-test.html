📋 Test Instructions

  1. Click "Initialize Terminal" - Creates the xterm instance
  2. Click in the terminal area - Should show focus state change
  3. Type some characters - Each keypress should be logged
  4. Try special keys - Enter, Backspace, Arrow keys, Ctrl+C, etc.
  5. Use "Test Keyboard" - Programmatically tests key capture
  6. Check Focus - Focus state should change when clicking in/out

🔍 What to Look For

  • Green terminal state - Terminal initialized successfully
  • Incrementing counters - Key presses and onData events should increase
  • Focus changes - Should show "Focused" when terminal is active
  • Detailed logs - Every event should be logged with full details

⚠️ Common Issues

  • No key events: xterm.js not properly loaded or initialized
  • Focus problems: Terminal container CSS issues or event bubbling
  • Character encoding: Special characters not handled correctly
  • Performance: Too many logs slowing down the browser
